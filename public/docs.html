功能

获取精选产品列表信息

URL

https://api.xueqiu.com/private_fund/v4/index/bzb/list.json

支持格式

json

HTTP请求方式

GET

是否需要身份验证

是

请求参数

参数名称 参数类型 是否必填 说明
risk_match boolean 否 是否风险匹配，false：展示全部，true：展示风险匹配产品，默认true
industry Integer 否 所属行业 1、医药 2、消费 3、互联网 4、大金融，不传返回全部，


注意事项

无
